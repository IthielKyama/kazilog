# Web Dashboard

React + Vite dashboard for KaziLog admins, supervisors, and assessors.

## Local Development

1. Copy `.env.example` to `.env`.
2. Run `npm run setup`.
3. Run `npm run dev`.

The dashboard reads its backend base URL from:

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

For the full monorepo setup flow, use the root-level instructions in the repository `README.md`.
